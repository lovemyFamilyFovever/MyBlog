# JavaScript 基础

JavaScript 是一门动态的脚本语言，可以嵌入到 HTML 页面中，为用户提供丰富的交互式的用户体验。

## 基本语法

JavaScript 语言的基本语法包括：

- 变量声明：`var`、`let`、`const`
- 数据类型：`Number`、`String`、`Boolean`、`Array`、`Object`
- 条件语句：`if`、`else`、`switch`
- 循环语句：`for`、`while`、`do-while`
- 函数：`function`
- 运算符：`+`、`-`、`*`、`/`、`%`、`++`、`--`、`+=`、`-=`、`*=`、`/=`、`%=`
- 注释：`//`、`/* */`
